exmlrpc by rwbr

An HTTP 1.1 compliant XML-RPC library for Erlang originally made by Joakim Grebenö.

For an Erlang based project I'm working on, I searched for an implementation of a XML-RPC library for Erlang. This repository contains the original code with additional fixes, enhancements and further developments.

Dependencies

Erlang xmerl
